# **App Name**: TrendifyAI

## Core Features:

- Trending Topics Retrieval: Fetch real-time trending topics from Reddit for a selected region. Supports caching to minimize API calls.
- AI-Powered Content Generation: Generate smart social media posts, image prompts, and video scripts tailored to the trending topic using AI. A tool decides if extra details should be incorporated.
- Regional Trend Filtering: Allow users to select a region to view localized trending topics.
- Trend Filtering: Allow users to filter topics via text-based search.
- Dynamic Visualizations: Display trend data using sparklines to visualize growth and volume.
- Cache Management: Implement cache management for trends to quickly load data.
- AI Agent Creation: Generate an AI agent to monitor and respond automatically

## Style Guidelines:

- Primary color: Vibrant lime green (#BCF50F) for a modern, tech-forward feel, reflecting the app's focus on trends and AI.
- Background color: Dark gray (#050505), nearly black, to provide a high-contrast backdrop that makes the bright primary color pop.
- Accent color: Slightly more yellow green (#99CC00). Used for secondary highlights.
- Body and headline font: 'Space Grotesk' (sans-serif) for a computerized, techy feel. Consider pairing it with 'Inter' for longer text blocks if needed.
- Code font: 'Source Code Pro' for displaying code snippets, especially AI generated code.
- Use modern, minimalist icons from Lucide to represent actions and categories, filled with the primary color for emphasis.
- Incorporate subtle animations using Framer Motion, such as scaling effects on interactive elements and smooth transitions when loading data or displaying AI-generated content.
